Dear Editors,

Please, find attached our manuscript "Bloch-McConnell Simulation Study – Comparison of Various CEST Simulation Frameworks", intended for publication as a Research Article in Magnetic Resonance in Medicine.

This paper reports a community benchmark in which twenty research groups simulated eight defined CEST scenarios using independent Bloch–McConnell frameworks. The study quantifies inter-implementation agreement, identifies major sources of variability (including MT modeling, shaped RF pulses, and numerical settings), and provides open benchmark data together with analysis scripts to support reproducible quantitative CEST simulation practice.

We confirm that this manuscript has not been published elsewhere and is not under consideration by another journal. 
All authors have approved the manuscript and agree with its submission to MRM.

We disclose that AI-assisted tools were used in the preparation of this submission, both for developing the Python analysis code and for drafting and editing the manuscript text. 

Because reproducibility is central to this work, we ask the journal for a code review as part of the assessment process. The analysis scripts and archived dataset are described and linked in the submission.
